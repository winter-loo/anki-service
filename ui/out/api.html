Anki API Lab

1. Authentication & Headers

Note: Bearer token is usually obtained via Supabase auth. For local dev, you can bypass auth if you modify the server or use a dummy token if auth is disabled.

2. General API

GET /api/auth/whoami
POST /api/public/signup

3. Collections

GET /api/collection
GET /api/collection/stats
GET /api/collections
POST /api/collections
POST /api/collections/{id}/init
DELETE /api/collections/{id}

4. Decks

GET /api/decks
GET /api/decks/current
GET /api/deck/stats
POST /api/decks
POST /api/decks/select/{id}
DELETE /api/decks/{id}

5. Notes

GET /api/note/list
GET /api/note/studied_today
POST /api/note/add/{fld}
POST /api/note/add
GET /api/note/@{note_id}
POST /api/note/update/@{note_id}
POST /api/note/delete/@{note_id}

6. Cards & Scheduling

GET /api/card/sched_timing_today
GET /api/card/next
POST /api/card/answer/{ease}
GET /api/card/stats/@{card_id}

7. Misc / AI

GET /ui-config.json
GET /explain/{text}
